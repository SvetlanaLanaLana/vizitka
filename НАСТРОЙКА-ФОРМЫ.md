# Как включить форму на GitHub Pages

Сайт на `github.io` **не поддерживает PHP**. Нужен бесплатный «посредник» для отправки в Telegram.

Выберите **один** способ (способ 1 проще, если нет аккаунта Vercel).

---

## Способ 1. Google Таблицы / Apps Script (рекомендуется)

### Шаг 1. Создайте скрипт
1. Откройте [script.google.com](https://script.google.com)
2. **Новый проект**
3. Удалите код по умолчанию
4. Скопируйте весь код из файла **`google-apps-script/Code.gs`** в этот проект

### Шаг 2. Сохраните секреты
1. В функции **`setupSecrets`** замените `ВАШ_ТОКЕН_ОТ_BOTFATHER` на ваш токен
2. Вверху выберите функцию **`setupSecrets`**
3. Нажмите **Выполнить** (▶)
4. Разрешите доступ Google (первый раз спросит права)

### Шаг 3. Опубликуйте веб-приложение
1. **Развернуть** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. Запуск от имени: **Я**
4. У кого есть доступ: **Все**
5. **Развернуть** → скопируйте **URL** (длинная ссылка на `script.google.com/.../exec`)

### Шаг 4. Вставьте URL в сайт
Откройте **`form-config.js`** и вставьте ссылку:

```js
var formApiUrl = 'https://script.google.com/macros/s/ВАШ_ID/exec';
```

### Шаг 5. Загрузите на GitHub
Закоммитьте и запушьте обновлённый `form-config.js`.

### Шаг 6. Проверка
1. Напишите [@zayavka_sveta_bot](https://t.me/zayavka_sveta_bot) → `/start`
2. Откройте сайт → отправьте тестовую заявку

3. **Environment Variables:**

   - `TELEGRAM_CHAT_ID` = `5232226311`
   - `TELEGRAM_BOT_USERNAME` = `zayavka_sveta_bot`
4. **Deploy** → скопируйте URL, например `https://vizitka-xxxx.vercel.app`
5. В **`form-config.js`:**
   ```js
   var formApiUrl = 'https://vizitka-xxxx.vercel.app/api/send-telegram';
   ```
6. Загрузите `form-config.js` на GitHub

## Если форма не отправляется

| Ошибка | Решение |
|--------|---------|
| «Укажите URL в form-config.js» | Не заполнен `formApiUrl` — см. шаг 4 выше |
| «can't initiate conversation» | Напишите боту `/start` |
| «Сервер API не ответил» | Проверьте URL; для Google — переразверните веб-приложение |
