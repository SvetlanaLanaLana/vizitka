# Заявки в Telegram на GitHub Pages

Сайт на `github.io` **не поддерживает PHP**.  
Чтобы заявки приходили **в Telegram**, нужен бесплатный API (один раз, ~5 минут).

---

## Способ 1. Google Script (рекомендуется)

### 1. Создайте скрипт
1. [script.google.com](https://script.google.com) → **Новый проект**
2. Удалите код по умолчанию
3. Вставьте код из файла **`google-apps-script/Code.gs`**

### 2. Укажите токен бота
В функции **`setupSecrets`** замените `ВАШ_ТОКЕН_ОТ_BOTFATHER` на ваш токен:
```
8568379396:AAE8zWrKYCrCd2cX7lt3Q3nvAjAmiIm1pic
```

### 3. Запустите setupSecrets
1. Вверху выберите функцию **`setupSecrets`**
2. Нажмите **Выполнить** (▶)
3. Разрешите доступ Google

### 4. Опубликуйте
1. **Развернуть** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. Запуск от имени: **Я**
4. Доступ: **Все**
5. **Развернуть** → скопируйте URL (`https://script.google.com/macros/s/.../exec`)

### 5. Вставьте URL в сайт
В **`form-config.js`**:
```js
var formApiUrl = 'https://script.google.com/macros/s/ВАШ_ID/exec';
```

### 6. Загрузите на GitHub
Запушьте `form-config.js` и обновите страницу (**Ctrl+F5**).

### 7. Проверка
1. [@zayavka_sveta_bot](https://t.me/zayavka_sveta_bot) → `/start`
2. Отправьте тестовую заявку с сайта
3. Сообщение придёт вам в Telegram

## Способ 2. Vercel

1. [vercel.com](https://vercel.com) → войти через GitHub
2. **Add New → Project** → репозиторий `vizitka`
3. **Environment Variables:**
   - `TELEGRAM_BOT_TOKEN` = `8568379396:AAE8zWrKYCrCd2cX7lt3Q3nvAjAmiIm1pic`
   - `TELEGRAM_CHAT_ID` = `5232226311`
   - `TELEGRAM_BOT_USERNAME` = `zayavka_sveta_bot`
4. **Deploy** → скопируйте URL проекта
5. В **`form-config.js`:**
   ```js
   var formApiUrl = 'https://vizitka-xxxx.vercel.app/api/send-telegram';
   ```
6. Загрузите на GitHub

---

## Ошибки

| Ошибка | Что делать |
|--------|------------|
| «нужен бесплатный API» | Не заполнен `formApiUrl` — см. шаг 5 |
| «can't initiate conversation» | Напишите боту `/start` |
| «API не ответил» | Проверьте URL; переразверните Google Script |
